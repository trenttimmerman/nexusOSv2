# 3D Asset Library: Design Studio Resource Guide

## Overview
This comprehensive library contains **free-to-use 3D assets** organized by category and use case. All assets are optimized for web use and compatible with modern section designs.

**Supported Formats:**
- `.splinecode` (Spline - Best for web, smallest file size)
- `.gltf` / `.glb` (Universal 3D format)
- `.obj` (Mesh only, no animations)
- `.usdz` (Apple AR Quick Look)

## üì¶ Asset Management Best Practices

### File Size Optimization
```
Target file sizes for web:
- Hero Section: < 2MB
- Feature Icons: < 200KB
- Background Elements: < 500KB
```

### Spline Export Settings
1. **File ‚Üí Export ‚Üí Spline**
   - ‚úÖ Compressed
   - ‚úÖ Remove unused assets
   - ‚ö†Ô∏è Reduce texture quality if > 1MB

2. **For React Integration:**
   ```bash
   npm install @splinetool/react-spline
   ```

### Performance Tips
- **Lazy Load:** Use `loading="lazy"` for below-fold 3D
- **Fallback Images:** Provide 2D PNG for slow connections
- **Mobile:** Replace heavy 3D with static images on mobile
- **Intersection Observer:** Start animation only when visible

## üí° Usage Examples by Section Type

### Hero Section
**Recommended:** Morphing Blob, Gradient Sphere, Liquid Glass Flower
**Position:** Background, floating offset right
**Animation:** Slow rotation (0.1 RPM), subtle floating

### Feature Cards (Bento Grid)
**Recommended:** 3D Icons, Geometric primitives, Product mockups
**Position:** Top-right corner of card
**Animation:** Bounce on hover, scale up 1.1x

### Testimonial Wall
**Recommended:** Abstract avatars, Emoji reactions
**Position:** Left side of quote card
**Animation:** Slight tilt on scroll

### Pricing Table
**Recommended:** Tiers represented by stacked objects (Bronze/Silver/Gold spheres)
**Position:** Above price tag
**Animation:** Glow effect on hover

### Footer
**Recommended:** Low-poly landscape, Particle field
**Position:** Full-width background
**Animation:** Gentle parallax on scroll

## üéØ Integration Code Template

### Basic Spline Component (React)
```jsx
import Spline from '@splinetool/react-spline';

export default function HeroSection() {
  return (
    <div className="relative h-screen">
      {/* 3D Background */}
      <div className="absolute inset-0 -z-10">
        <Spline scene="https://prod.spline.design/YOUR-SCENE-ID/scene.splinecode" />
      </div>
      
      {/* Content Overlay */}
      <div className="relative z-10 flex items-center justify-center h-full">
        <h1 className="text-6xl font-bold text-white">Your Headline</h1>
      </div>
    </div>
  );
}
```

### With Loading State
```jsx
import { Suspense } from 'react';
import Spline from '@splinetool/react-spline';

function Hero() {
  return (
    <Suspense fallback={<LoadingSkeleton />}>
      <Spline scene="URL" />
    </Suspense>
  );
}

function LoadingSkeleton() {
  return (
    <div className="animate-pulse bg-gray-200 w-full h-full" />
  );
}
```
